The Concept

Concept of APIStarter.com: kickstarter for API.

Crowd source funding for API libraries. Pay $2 bucks for the library you need, get 50 backers, and we will find a dedicated developer to implement, test, and document the library in a short time frame. This is the concept that I think IT will move forward to in the near future.
